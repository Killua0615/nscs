プライバシーポリシー

最終更新日: 2026年7月8日

1. 収集する情報

本アプリ「NSCA-CPT 試験対策」は、学習の進捗データ（解答履歴・理解度ステータス）をデバイス内にのみ保存します。個人を特定できる情報（氏名・メールアドレス等）は収集しません。

2. データの保存場所

学習データはお使いのデバイス内（AsyncStorage）に保存されます。「間違いを報告」機能をご利用の場合、入力された報告内容がGoogle フォームを通じて送信されます。

3. 第三者への提供

本アプリは利用者のデータを第三者に提供・販売することはありません。

4. 分析ツール

本アプリは現時点で分析ツール（アナリティクス）を使用していません。将来的に導入する場合は、本ポリシーを更新いたします。

5. お子様のプライバシー

本アプリは13歳未満のお子様を対象としておりません。

6. お問い合わせ

プライバシーに関するお問い合わせは、App Store のアプリページよりご連絡ください。

7. 変更について

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載されます。
